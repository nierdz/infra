Faire fonctionner Xen et virt-manager

Écrit le 29 12 2012

Une petite astuce pour faire fonctionner virt-manager avec Xen. Vous devez éditer le fichier de conf de Xen :


# vim /etc/xen/xend-config.sxp

Et vous devez y ajouter ceci : 


(xend-unix-server yes)

Ça permet de démarrer une socket de management de xend.

Ensuite, vous devez redémarrer xen : 


# /etc/init.d/xend restart

En ce qui concerne le client qui utlise virt-manager, vous devez installer le paquet ssh-askpass qui va vous permettre de rentrer votre mot de passe pour établir une connexion SSH vers votre serveur xen.
