Benchmark EG 64G Hybrid (SSD + SATA3) 2013 OVH

Écrit le 13 06 2013

Je vais faire une série d'articles qui n'en sont pas vraiment mais qui peuvent être utile et servir de référence si vous souhaitez louer un serveur chez OVH. On commence par la machine EG 64G Hybrid de chez OVH.

Caractéristiques techniques et informations supplémentaires

  • OS : Debian Squeeze, kernel OVH 3.8.13-xxxx-grs-ipv6-64
  • Disque : SSD INTEL SSDSA2CW120G3 (2 disques)
  • Raid : Raid1 avec les paramètres par défaut OVH
  • CPU : Intel(R) Xeon(R) CPU E5-1620 0 @ 3.60GHz
  • RAM : 64Go
