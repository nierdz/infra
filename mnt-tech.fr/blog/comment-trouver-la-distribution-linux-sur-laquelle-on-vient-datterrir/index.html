Comment trouver la distribution Linux sur laquelle on vient d'atterrir

Écrit le 14 07 2012

Si en arrivant sur un serveur vous êtes un peu perdu et que vous souhaitez savoir sur quel version de linux vous êtes, il y a plusieurs fichiers qui peuvent vous aider :

# cat /etc/issue

exemple de sortie sur un kimsufi avec une Debian :


Linux ksXXXXX.kimsufi.com 2.6.32-5-xen-amd64 #1 SMP Sun May 6 08:57:29 UTC 2012 x86_64 GNU/Linux
server    : XXXXX
ip        : 188.165.XX.XX
hostname  : ksXXXXX.kimsufi.com

# cat /etc/motd

exemple de sortie, toujours sur un kimsufi avec une Debian :


Linux ksXXXXX.kimsufi.com 2.6.32-5-xen-amd64 #1 SMP Sun May 6 08:57:29 UTC 2012 x86_64 GNU/Linux
server    : XXXXX
ip        : 188.165.XX.XX
hostname  : ksXXXXX.kimsufi.com

# cat /etc/debian_version

exemple de sortie sur une debian squeeze :


6.0.5

# cat /etc/redhat-release

exemple de sortie sur une CentOS 5.3 :


CentOS release 5.3 (Final)

Vous pouvez également utiliser :

# cat /etc/lsb-release

Qui peut renvoyer un truc du genre sur une Debian :


DISTRIB_ID=Debian
DISTRIB_RELEASE=7
DISTRIB_CODENAME=
DISTRIB_DESCRIPTION=

Mais attention, d'une manière générale ne vous fiez pas à toutes ces informations qui peuvent être érronées si la distribution a été mise à jour... Bref, le mieux est de faire confiance uniquement au gestionnaire de paquet et aux fichiers qui en dépendent. Par exemple, sur Debian, le plus fiable est de consulter le fichier /etc/apt/sources.list.
