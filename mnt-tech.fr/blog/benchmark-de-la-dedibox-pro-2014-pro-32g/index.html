Benchmark de la dedibox Pro 2014 (Pro 32G)

Écrit le 05 12 2013

Aujourd'hui on va se faire un petit benchmark. Il s'agit cette fois d'une machine online pour changer un peu. Plus précisément il s'agit de la Dedibox Pro 2014 en 32G qui est en fait un HP DL320e v2. Cette machine coute 69.99€ HT / mois et est plus ou moins équivalente à la SP bestof juillet 2013. Elle est tout de même un peu mieux niveau CPU et IO comme vous allez le voir dans les benchmarks suivants. Vous pouvez choisir le type de disques durs et vous avez le choix entre du SSD, du SAS ou du SATA. Dans notre cas, nous avions besoin de place, nous avons opté pour du SATA. Elle est vendue avec un vlan et est donc plutôt performante pour une machine dans le bas de la moyenne gamme. Avec du SATA ,elle peut donc être utilisée pour du LAMP avec un trafic moyennement important ou dans une architecture hadoop ou cloud.

Je viens de mettre à jour l'article de la synthèse des benchmarks

Caractéristiques techniques et informations supplémentaires

  • OS : Debian Wheezy, kernel debian 3.2.0-4-amd64
  • Disque : HP 3To MB3000GCWLU
  • Raid : Raid1 sur carte Smart Array P222
  • CPU : Intel(R) Xeon(R) CPU E3-1240 v3 @ 3.40GHz
  • RAM : 32Go
